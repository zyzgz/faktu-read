# Faktu-Read

## Client

This directory contains the Angular project.

## Server

This directory will contain the server-side code in the future.

## Setup

### Client

1. Navigate to the `client` directory:
    ```sh
    cd client
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Run the Angular project:
    ```sh
    ng serve
    ```
